# prts-micro-frontends

## setup for development
```sh
git clone https://github.com/MooncellWiki/prts-micro-frontends.git
cd prts-micro-frontends
pnpm i
pnpm run dev
```
提交时运行
```sh
pnpm run commit
git push
```
会逐步迁移一些没人会去搜，而且需要复杂lua/smw查询的模板过来
